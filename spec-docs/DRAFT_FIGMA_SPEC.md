## Design Principles

1. **Excitement**: Draft should feel like a real event with anticipation
2. **Information Rich**: Players need stats to make informed decisions
3. **Clear State**: Always obvious whose pick it is and what's available
4. **Efficiency**: Support both deliberate selection and quick auto-draft
5. **Farm Focus**: Always show how pick affects Farm roster needs
6. **Potential Display**: Show both current grade AND potential ceiling
