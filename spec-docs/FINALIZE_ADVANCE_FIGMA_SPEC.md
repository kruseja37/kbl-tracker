## Design Principles

1. **Clarity**: Always show current roster state (X/22 MLB, Y/10 Farm)
2. **Actionable**: Clear call-to-action buttons for roster changes
3. **Validation-Driven**: Cannot advance until all rosters are valid
4. **SMB4 Sync**: Transaction report designed for easy replication in game
5. **Progressive Disclosure**: Guide user through steps without overwhelming
