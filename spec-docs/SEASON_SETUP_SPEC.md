# Season Setup Specification

> **Purpose**: Define the flow when user clicks "New Franchise" or "Playoff Mode"
> **Created**: January 29, 2026
> **Status**: DRAFT
> **Related**: LEAGUE_BUILDER_SPEC.md, FRANCHISE_MODE_SPEC.md

---

## 1. Overview

Season Setup is the wizard that appears when a user wants to start a new franchise or playoff series. It takes configurations from the League Builder and allows the user to customize them for this specific playthrough.

### 1.1 Entry Points

| Entry | Flow | Description |
|-------|------|-------------|
| **New Franchise** | Full wizard | Complete season setup with all options |
| **Playoff Mode** | Abbreviated | Skip season settings, go straight to playoffs |
| **Continue Franchise** | None | Resumes existing franchise (no setup needed) |

## 6. Step 4: Team Control

### 6.1 Purpose

Select which teams the user(s) will control. Non-selected teams become AI-controlled.

### 6.2 Key Features

- **Multi-select** - User can control 1 to all teams
- **Sticky selection** - Buttons/checkboxes remain selected when clicked
- **Visual feedback** - Clear distinction between user and AI teams
- **Multiplayer support** - Multiple users can control different teams

## 11. Navigation & State

### 11.1 State Management

```typescript
interface SetupWizardState {
  currentStep: 1 | 2 | 3 | 4 | 5 | 6;

  validationErrors: Record<number, string[]>;
```

### 11.2 Navigation Rules

- **Back**: Always available (except Step 1)
- **Next**: Only enabled when current step is valid
- **Cancel**: Prompts "Are you sure?" confirmation
- **Jump to Step**: Can jump back to completed steps, cannot skip ahead
