# Free Agency UI/UX Design Specification

> **For**: Figma Design Team
> **App**: KBL XHD Tracker (iPad-first baseball league management)
> **Feature**: Offseason Free Agency System
> **Created**: January 28, 2026

---

## Overview

Free Agency is a **high-drama offseason feature** where players can leave teams based on dice rolls and personality-driven destinations. The UI should feel like a combination of:
- **Draft night tension** (dice rolls, reveals)
- **Game show excitement** (animations, reveals)
- **Sports app clarity** (player cards, stats)

### Design Principles
1. **Drama & Anticipation** - Build tension with animations and reveals
2. **Clarity** - Always show what's happening and why
3. **Touch-First** - Large tap targets for iPad (44pt minimum)
4. **Baseball Aesthetic** - Team colors, card-based player displays

### Component Specs

**Dice**
- Size: 80pt Ã— 80pt
- 3D appearance with shadow
- Pips clearly visible
- Animation: tumble/bounce for 1.5 seconds

**Dice Table Row**
- Drag handle (â˜°): 32pt width, gray color, cursor: grab
- Dice value column: 40pt width, centered, bold
- Probability bar: Proportional width, team color fill
- Highlighted row (post-roll): Yellow/gold background, scale up slightly
- Row height: 48pt minimum for touch targets

**Reorder Controls**
- "Reset to Default" button: Top-right of table, secondary style
- Hint text: "Drag rows to reorder. Top = highest risk."
- Drag feedback: Row lifts with shadow, drop zone highlighted
- Dice values stay fixed to positions (position 1 always = dice 7)

**Result Card**
- Player photo: 96pt Ã— 96pt
- Name: 24pt bold
- Position/Grade/Personality: 16pt
- Personality shown with icon (see personality icons below)

### Interactions

**Pre-Roll (Reorder Phase)**
1. **Drag row**: Touch and hold drag handle (â˜°), move to new position
2. **Drop row**: Release to place in new order
3. **Reset**: Tap "Reset to Default" to restore grade-based auto-sort
4. **Roll**: Tap "ROLL DICE" to commit order and roll

**Post-Roll (Result Phase)**
- No reorder allowed after roll
- Matched row highlighted
- Continue to destination

### Animations
1. **Pre-roll**: Dice have subtle "ready" wobble
2. **Drag**: Row lifts, other rows shift to show drop zone
3. **Rolling**: Dice tumble with physics (1.5s)
4. **Landing**: Dice settle, sum appears
5. **Reveal**: Table row highlights with pulse animation
6. **Card slide**: Result card slides up from bottom

## Screen 4: Player Exchange

### Purpose
When a player moves to a new team, the receiving team must give back a player of comparable **salary value** (Â±10%).

### Core Rule: Salary-Based Matching
- Return player must be within **Â±20% of incoming player's TRUE VALUE (salary)**
- **No position matching required** - any player can be exchanged for any player (teams can draft/call-up replacements)
- **Fallback**: If no one meets threshold, must give the player whose salary is **CLOSEST to the incoming player's salary**
