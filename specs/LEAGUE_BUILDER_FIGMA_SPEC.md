## 14. Interaction Patterns

### 14.1 Drag and Drop

| Context | Source | Target | Action |
|---------|--------|--------|--------|
| Lineup | Bench player | Lineup slot | Swap or insert |
| Lineup | Lineup player | Lineup slot | Reorder |
| Roster | MLB roster | Farm roster | Send down |
| Roster | Farm roster | MLB roster | Call up |
| Roster | Free agents | Team roster | Sign |
| Teams | Team card | Division | Assign to division |
| Draft | Available player | Team slot | Draft pick |

### 14.2 Sticky Toggle Buttons

Used in: Team Selection (Season Setup Step 4)

- Default: Outline button, secondary color
- Selected: Filled button, primary color, check icon
- Click toggles between states
- Visual feedback: slight scale animation on toggle

### 14.3 Slider Controls

- Drag thumb or click track to set value
- Value tooltip appears on drag
- Keyboard: Arrow keys adjust by 1, Shift+Arrow by 10
- Color coding optional (green/yellow/red zones)
